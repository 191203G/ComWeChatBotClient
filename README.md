## 简介

`ComWeChatRobot`的客户端封装，支持`onebot12`通信协议。

## 上游依赖

- [ComWeChatRobot](https://github.com/ljc545w/ComWeChatRobot)：PC微信机器人，实现获取通讯录，发送文本、图片、文件等消息，封装COM接口供Python、C#调用
